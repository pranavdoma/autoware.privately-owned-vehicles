# Quantization Exports

Scripts are grouped by quantization workflow, then model name:

- `PTQ/AutoDrive/autodrive_ptq.py` — AutoDrive PT2E post-training quantization.
- `QAT/AutoDrive/autodrive_qat.py` — AutoDrive QAT checkpoint export and benchmark.
- `QAT/SceneSeg/benchmark_qat_model.py` — SceneSeg QAT conversion and benchmark.
- `QAT/SceneSeg/quantize_model_sceneseg.py` — SceneSeg QAT training/export helper.
